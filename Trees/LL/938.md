### Optimized BST Traversal
1. **BST Property Utilization**: Leverages BST structure to skip irrelevant subtrees
2. **Recursive Depth-First Search**:
   - If current node < low â†’ only check right subtree
   - If current node > high â†’ only check left subtree
   - Else â†’ add current node value and check both subtrees

## ğŸ’» Code Implementation

```go
/**
 * Definition for a binary tree node.
 * type TreeNode struct {
 *     Val int
 *     Left *TreeNode
 *     Right *TreeNode
 * }
 */
func rangeSumBST(root *TreeNode, low int, high int) int {
    if root == nil {
        return 0
    }

    // Use BST properties to prune search
    if root.Val < low {
        return rangeSumBST(root.Right, low, high)
    }
    if root.Val > high {
        return rangeSumBST(root.Left, low, high)
    }

    // Current node is in range
    return root.Val + 
           rangeSumBST(root.Left, low, high) + 
           rangeSumBST(root.Right, low, high)
}
```
